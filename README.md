# clust431

<!-- badges: start -->

<!-- badges: end -->

The goal of clust431 is to do k-means clustering and hierarchical
clustering.

## Installation

You can install the released version of clust431 from
[CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("clust431")
```
